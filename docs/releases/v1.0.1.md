# Agent Signal Bar v1.0.1

## English

### Highlights

- Polished the first-run settings and menu surfaces so the app feels less like a developer tool and more like a usable menu bar utility.
- Added visual README assets that show the app icon, menu bar panel, and status bar behavior.
- Improved settings controls and labels around appearance, effects, connections, and general app actions.

### Changes

- Added app icon and menu bar panel screenshots to the README.
- Highlighted customizable light effects, multilingual UI, and local-first agent integration in the documentation.
- Refined the menu bar panel quick actions and settings-window layout.
- Improved status bar and settings copy across supported languages.
- Bumped app and release metadata to 1.0.1.

### Fixes

- Fixed settings and menu panel layout inconsistencies around action buttons and dropdown controls.
- Improved status bar controller behavior around opening and closing the settings window.

### Release verification

- GitHub Actions built the app, ran Swift tests, packaged the release, and verified SHA-256 checksums.
- Checksums are published in [AgentSignalLight-SHA256SUMS.txt](https://github.com/guan-ops/Agent-Signal-Bar/releases/download/v1.0.1/AgentSignalLight-SHA256SUMS.txt).
- Release metadata is published in [AgentSignalLight-release-manifest.json](https://github.com/guan-ops/Agent-Signal-Bar/releases/download/v1.0.1/AgentSignalLight-release-manifest.json).
- This is a local/self-use macOS build. It is ad-hoc signed and not notarized.

- README 新增 App 图标和状态栏小菜单截图。
