# Agent Signal Bar v1.0.4

## English

### Highlights

- Refined the settings window with a more unified Liquid Glass look across the titlebar, header, tab bar, content area, dropdowns, and segmented controls.
- Added a reduced Liquid Glass strength and made it the default for fresh installs.
- Refreshed README screenshots for the Liquid Glass settings window and menu bar panel.

### Changes

- Renamed the settings page from `Appearance` to `Style` and updated the localized copy.
- Renamed glass-effect wording to Liquid Glass across the app and README.
- Added reduced, standard, and enhanced Liquid Glass behavior for the settings window.
- Improved dropdown and segmented-control sizing so settings controls align more consistently.
- Updated the Chinese menu bar panel screenshot and README Liquid Glass screenshots.
- Bumped app and release metadata to 1.0.4.

### Fixes

- Fixed the macOS app menu `Settings...` command so it opens the Agent Signal Bar settings window.
- Fixed an intermittent issue where clicking the menu bar signal light could also open the settings window.
- Improved settings window chrome so the titlebar blends with the app content.
- Fixed settings tab-bar color mismatch against the rest of the window.

### Release verification

- GitHub Actions built the app, ran Swift tests, packaged the release, and verified SHA-256 checksums.
- Checksums are published in [AgentSignalLight-SHA256SUMS.txt](https://github.com/guan-ops/Agent-Signal-Bar/releases/download/v1.0.4/AgentSignalLight-SHA256SUMS.txt).
- Release metadata is published in [AgentSignalLight-release-manifest.json](https://github.com/guan-ops/Agent-Signal-Bar/releases/download/v1.0.4/AgentSignalLight-release-manifest.json).
- This is a local/self-use macOS build. It is ad-hoc signed and not notarized.

## 简体中文

### 亮点

- 优化设置窗口的液态玻璃视觉，让标题栏、顶部区域、菜单栏、内容区、下拉框和分段选择控件更统一。
- 新增液态玻璃 `减弱` 强度，并将新安装默认设置为减弱。
- 更新 README 中的液态玻璃设置窗口和状态栏小菜单截图。

### 改进

- 设置页名称从 `外观` 改为 `样式`，并同步更新多语言文案。
- 将 App 和 README 中的毛玻璃相关文案统一改为液态玻璃。
- 设置窗口支持减弱、标准、增强三种液态玻璃显示强度。
- 优化下拉框和分段选择控件尺寸，让设置项更整齐。
- 更新中文状态栏小菜单截图和 README 液态玻璃截图。
- 将 App 和发布元数据更新到 1.0.4。

### 修复

- 修复 macOS 应用菜单里的 `Settings...` 无法正常打开 Agent Signal Bar 设置窗口的问题。
- 修复点击状态栏信号灯时，偶尔会误打开设置窗口的问题。
- 优化设置窗口标题栏，使其与应用内容更像一个整体。
- 修复设置窗口菜单栏与其他区域颜色不一致的问题。
