# Agent Signal Bar v1.0.0

## English

### Highlights

- First public release of Agent Signal Bar, a local-first macOS menu bar app for watching AI agent state at a glance.
- Added the red/yellow/green signal language for idle, thinking, working, attention, done, permission, blocked, stale, paused, and off states.
- Shipped the first local release package with a DMG, ZIP archive, release manifest, and checksum file.

### Changes

- Added the macOS app with menu bar signal light, settings window, menu bar panel, and local state monitoring.
- Added horizontal and vertical signal layouts plus classic signal-board and minimal-dot styles.
- Added Codex Desktop monitoring, Codex hook support, Claude Code hook support, and a generic JSON event adapter for local scripts.
- Added the `agent-signal` CLI and `agent-signal-run` wrapper for scripts and automation.
- Added session aggregation so red and yellow states keep priority over normal working states.
- Added multilingual UI, theme selection, launch-at-login support, diagnostics, release packaging, and local verification scripts.

### Fixes

- Fixed clean-runner release packaging so GitHub Actions can publish installable artifacts from a fresh checkout.
- Improved CI stability for Swift tests and icon rendering checks.

### Release verification

- GitHub Actions built the app, ran Swift tests, packaged the release, and verified SHA-256 checksums.
- Checksums are published in [AgentSignalLight-SHA256SUMS.txt](https://github.com/guan-ops/Agent-Signal-Bar/releases/download/v1.0.0/AgentSignalLight-SHA256SUMS.txt).
- Release metadata is published in [AgentSignalLight-release-manifest.json](https://github.com/guan-ops/Agent-Signal-Bar/releases/download/v1.0.0/AgentSignalLight-release-manifest.json).
- This is a local/self-use macOS build. It is ad-hoc signed and not notarized.

## 简体中文

### 亮点

- Agent Signal Bar 首个公开版本：一个本地优先的 macOS 状态栏应用，用来一眼查看 AI Agent 当前状态。
- 新增红、黄、绿灯语，覆盖空闲、思考中、工作中、需要查看、完成、请求授权、阻塞、过期、暂停和关闭灯状态。
- 首次提供本地安装包，包括 DMG、ZIP、版本清单和校验文件。

### 改进

- 新增 macOS App、状态栏信号灯、设置窗口、状态栏小菜单和本地状态监听。
- 新增横向 / 竖向布局，以及经典灯牌 / 极简圆点两种状态栏样式。
- 新增 Codex Desktop 监控、Codex Hook、Claude Code Hook，以及给本地脚本使用的通用 JSON 事件接入。
- 新增 `agent-signal` CLI 和 `agent-signal-run` 包装命令，方便脚本和自动化流程上报状态。
- 新增多会话聚合逻辑，确保红灯和黄灯优先级不会被普通工作状态覆盖。
- 新增多语言 UI、主题选择、开机自启动、诊断导出、发布打包和本地验证脚本。

### 修复

- 修复干净 GitHub Actions runner 上的发布打包问题，确保可以从全新 checkout 生成安装包。
- 提升 Swift 测试和图标渲染检查在 CI 中的稳定性。

### 发布验证

- GitHub Actions 已完成 App 构建、Swift 测试、发布打包和 SHA-256 校验。
- 校验值见 [AgentSignalLight-SHA256SUMS.txt](https://github.com/guan-ops/Agent-Signal-Bar/releases/download/v1.0.0/AgentSignalLight-SHA256SUMS.txt)。
- 版本清单见 [AgentSignalLight-release-manifest.json](https://github.com/guan-ops/Agent-Signal-Bar/releases/download/v1.0.0/AgentSignalLight-release-manifest.json)。
- 这是本地 / 自用 macOS 构建，使用 ad-hoc 签名，尚未进行 notarization 公证。
