# Agent Signal Bar v1.0.2

## English

### Highlights

- Expanded the README into a more visual product page with localized screenshots, real icon previews, and animated light-effect demos.
- Replaced the DMG README markdown file with a bilingual plain-text README for better readability after mounting the installer.
- Documented GitHub Release installation clearly so users download the app artifacts instead of the source ZIP.

### Changes

- Added localized English and Chinese settings screenshots.
- Added real menu bar panel screenshots for both README languages.
- Added animated light-effect previews generated from the actual status icon renderer.
- Added a dedicated `AgentSignalIconPreview` target for producing real status icon previews.
- Added standard glass settings-window screenshots and a clearer status bar demo layout.
- Documented the current integration verification status: Codex was tested, while Claude support is provided through hooks and still needs practical verification.
- Updated the release installer README to `docs/DMG_README.txt` with Chinese and English instructions.
- Bumped app and release metadata to 1.0.2.

### Fixes

- Fixed README visual spacing so classic signal-board and minimal-dot styles are easier to compare.
- Fixed release-install verification paths for the bilingual text README.

### Release verification

- GitHub Actions built the app, ran Swift tests, packaged the release, and verified SHA-256 checksums.
- Checksums are published in [AgentSignalLight-SHA256SUMS.txt](https://github.com/guan-ops/Agent-Signal-Bar/releases/download/v1.0.2/AgentSignalLight-SHA256SUMS.txt).
- Release metadata is published in [AgentSignalLight-release-manifest.json](https://github.com/guan-ops/Agent-Signal-Bar/releases/download/v1.0.2/AgentSignalLight-release-manifest.json).
- This is a local/self-use macOS build. It is ad-hoc signed and not notarized.
