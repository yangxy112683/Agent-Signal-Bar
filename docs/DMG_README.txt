Agent Signal Bar DMG Quick Start
================================

English
-------

1. Drag AgentSignalLight.app to Applications.
2. Open Agent Signal Bar from Applications.
3. If macOS says the developer cannot be verified, right-click the app and choose Open, or go to System Settings > Privacy & Security > Open Anyway.
4. Use General > Start at login if you want the app to open automatically after signing in.
5. Open Connect and click Install Connections if you want Codex and Claude Code events to drive the status light.
6. Use Copy Integration Command for local scripts or other agents that can emit JSON events.
7. Use Connect > Check or Export Diagnostics when troubleshooting.

This local DMG is ad-hoc signed for personal use. It is not Developer ID signed or notarized yet.

中文
----

1. 将 AgentSignalLight.app 拖到 Applications。
2. 从 Applications 打开 Agent Signal Bar。
3. 如果 macOS 提示无法验证开发者，请右键 App 选择“打开”，或进入“系统设置 > 隐私与安全性 > 仍要打开”。
4. 如果希望登录 macOS 后自动启动，可以在“通用 > 开机自启动”中开启。
5. 如果希望 Codex 和 Claude Code 事件驱动状态灯，请打开“连接”并点击“安装连接”。
6. 本地脚本或其他可以输出 JSON 事件的 Agent，可以使用“复制接入命令”。
7. 排查问题时，可以使用“连接 > 检查连接”或“导出诊断”。

这个本地 DMG 目前使用 ad-hoc 签名，适合个人使用，尚未使用 Developer ID 签名或 Apple 公证。
