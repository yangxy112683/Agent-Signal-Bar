# Agent Signal Bar DMG Quick Start

1. Drag `AgentSignalLight.app` to `Applications`.
2. Open the app from Applications.
3. Use the menu bar item to turn on `开机自启动` if you want it to start automatically.
4. Open `连接` and click `安装连接` if you want Codex and Claude Code events to drive the status light.
   Use `复制接入命令` for local runners or other agents that can emit JSON events.
5. Use `连接` > `检查连接` or `导出诊断` when troubleshooting.

This local DMG is ad-hoc signed for personal use. It is not Developer ID signed or notarized yet.
